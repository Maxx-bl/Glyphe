À propos de la version Web

La version web est une version de test et n'est pas entièrement supportée. Certaines fonctionnalités comme l'envoi de pièces jointes dans les messages ne sont pas encore fonctionnelles, et d'autres bugs peuvent subvenir. Privilégiez l'application Android pour une expérience optimale.

Comment installer l'application Android

Étant donné que Glyphe n'est pas encore sur le Play Store, voici les étapes pour l'installer en toute sécurité.

1

Autoriser les sources inconnues

Lors de l'ouverture de l'APK, votre téléphone peut bloquer l'installation. Allez dans les Paramètres > Sécurité et activez l'installation d'applications issues de sources inconnues.

2

Avertissement Play Protect

Google Play Protect affichera un message d'alerte car l'application est nouvelle. Cliquez sur "Plus de détails", puis sélectionnez "Installer quand même".
